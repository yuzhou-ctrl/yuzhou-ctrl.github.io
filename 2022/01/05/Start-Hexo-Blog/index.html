用hexo搭建博客(一)

发表于2022-01-05|更新于2022-01-05|-Hexo环境配置（一）
|阅读量:|评论数:

一、准备工作

  1. 新建一个文件夹并命名为Blog，最好在磁盘的根目录下，不然后面可能会涉及到改代码。（绝对路径和相对路径问题）

  2. 安装git环境

  3. 注册GitHub账号

  4. 安装node.JS

  5. 找到你新建的Blog文件夹，并在路径下右键git bash here打开命令提示框

  6. 安装hexo框架

    $ npm install -g hexo-cli

  7. 通过以下命令查看是否安装node.js和npm，gitbash不行的话可以在cmd命令行内，实在不行请检查系统环境变量是否正确配置。

二、正式开始

首先了解一下什么是Hexo三连，因为这个命令会经常用到，请确保你正确的输入

  1. 进入Blog文件夹下打开git_bash。
  2. 输入初始化命令,生成必要文件。
    hexo init
  3. 然后hexo三连，会发现输出了一个带端口号的网址，复制到浏览器，打开应该就能看到你刚刚弄好的博客啦。退出按CTRL+C没错就是你熟悉的那个操作。

至此，一个属于你的本地博客框架就搭好啦，后面会继续更新个性化博客功能，以及将博客上传到Github实现在线访问等操作。
