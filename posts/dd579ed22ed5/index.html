hexo搭建（一）

1. 环境支持：

Hexo 基于 Node.js，搭建过程中还需要使用 npm（Node.js 已带） 和 git，因此先搭建本地操作环境。

  • node 下载链接
  • Git 下载链接

环境验证：
分别输入以下命令验证安装版本，若失败请重新安装或寻找问题原因。

2. 配置Github

完成这步请确保你有GitHub账号

  1. 在文件管理器任意打开一个文件夹，右键
  2. 找到git bash here（别按错）会弹出命令行
  3. 输入以下命令
git config --global user.name "GitHub 用户名"
git config --global user.email "GitHub 邮箱"
  1. 创建SSH链接，这部保证你以后不用每次上传都需要密码。
    以下代码,一路回车就行，可能会让你设定文件名和密码，建议不理。
ssh-keygen -t rsa -C "GitHub 邮箱"
  1. 添加密钥，在GitHub网页设置找到对应界面，输入以下目录文件id_rsa.pub内的内容。
C:\Users\你的用户名\.ssh
  1. 验证链接,确认输入yes，若显示成功就行了
ssh -T git@github.com

3.安装hexo

如果git bash中不行请回到cmd命令行，推测大概率是没有配置好环境变量。

npm install -g hexo-cli

依次输入初始化和启动命令

hexo init      
npm install    
hexo g   
hexo s

会生成一个地址，复制到浏览器就可以看到对应网页了。
